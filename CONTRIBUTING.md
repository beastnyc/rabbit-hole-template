# Contributing

Thanks for wanting to improve Rabbit Hole Template.

## Local Setup

```bash
npm install
npm run build
npm run preview
```

## Before Opening A Pull Request

- Keep the template generic.
- Do not add private notes, personal vault paths, API keys, or deployment secrets.
- Keep sample content small and reusable.
- Run `npm run build`.

## Good Contributions

- clearer setup docs
- safer publishing guardrails
- accessibility fixes
- small UI improvements
- bug fixes for note linking, previews, search, or builds

## Not In Scope

- hosted accounts
- private-vault syncing
- server-side databases
- automatic publishing from an entire private vault
